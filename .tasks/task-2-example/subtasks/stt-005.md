Title: feature / Channels list — table view — handle missing usernames
Agent: Code Implementer

Goal
- Implement the channels list page with a simple table and clear handling for channels without public usernames.

Tech Requirements
- Technologies: React, SWR, TypeScript, Tailwind
- Endpoint: `GET /channels`
- Display columns: username (clickable if present), title, addedAt
- Non-clickable rows when `username = null` with hint text
- Loading/empty/error states with retry (SWR revalidate)
- Size: ≤ 400 LOC
- Dependencies: stt-003, stt-004

Acceptance Criteria
- Page fetches and renders channels in a table
- Rows for channels without username are visually disabled and non-navigable, with a hint
- Loading skeletons show while fetching; empty state and error with Retry are present
- Clicking a row with a username navigates to `/channels/:username`
