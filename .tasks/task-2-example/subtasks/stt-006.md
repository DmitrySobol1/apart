Title: feature / Posts list — date filters + pagination — dedupe on append
Agent: Code Implementer

Goal
- Build the channel posts list with local date-range filters, offset pagination, and deduplication when loading more.

Tech Requirements
- Technologies: React, SWR, TypeScript, Tailwind
- Endpoint: `GET /channels/:username/posts?from&to&limit&offset`
- Controls: `<input type="date">` for From/To; default last 30 days when absent
- Date handling: local day start/end → convert to UTC ISO for API
- Pagination: limit/offset; append results; dedupe by `messageId`
- Sorting: enforce date desc; preserve items across revalidation
- URL params: persist `from`, `to`, `limit`, `offset`
- Size: ≤ 500 LOC
- Dependencies: stt-003, stt-004

Acceptance Criteria
- Posts table renders with date, snippet/title, media icon, and latest counters
- Applying filters updates URL and refetches; Reset restores defaults
- Load more appends without duplicates; sort order remains date desc
- Revalidation retains accumulated items until new data arrives
