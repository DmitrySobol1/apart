## feature / Metrics endpoint — hourly series — 1y clamp at query time

### Goal
Provide a per-post hourly metrics endpoint with query-time clamping to the last 365 days.

### Tech Requirements
- Libraries: Hono, Drizzle ORM, `zod`/`drizzle-zod`.
- Depends on: stt-001 (schemas), stt-005 (posts route base).
- Size constraints: ≤3 files, ≤200 LOC.

### Acceptance Criteria
- `GET /channels/:username/posts/:messageId/metrics?from&to&bucket=hour` returns points validated by `PostMetricsPointSchema`.
- Query validation with `MetricsQuerySchema`; default range clamps to [post.date, now] and ≤365 days.
- Verification commands:
  - `pnpm --filter @tg-channel-stats/backend type-check`
  - `pnpm --filter @tg-channel-stats/backend build`
- Artifacts:
  - `packages/backend/src/routes/metrics.ts`
