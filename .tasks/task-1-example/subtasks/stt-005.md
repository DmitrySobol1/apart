## feature / Read endpoints — channels + posts list — DTOs via drizzle-zod

### Goal
Expose read APIs to list channels and channel posts with filters using DTOs derived from Drizzle via drizzle-zod.

### Tech Requirements
- Libraries: Hono, Drizzle ORM, `zod`/`drizzle-zod`.
- Depends on: stt-001 (schemas), stt-004 (optional route composition).
- Size constraints: ≤5 files, ≤300 LOC.

### Acceptance Criteria
- `GET /channels` returns channel rows or a picked subset validated via Zod.
- `GET /channels/:username/posts?limit&offset&from&to` returns list items validated by `PostListItemSchema`.
- Verification commands:
  - `pnpm --filter @tg-channel-stats/backend type-check`
  - `pnpm --filter @tg-channel-stats/backend build`
- Artifacts:
  - `packages/backend/src/routes/channels.ts`
  - `packages/backend/src/routes/posts.ts`
