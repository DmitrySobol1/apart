## Goal

Initialize the monorepo project structure with frontend and backend packages, environment configuration, and development tooling.

## Details

- Set up `frontend/` with Vite + React 18 + TypeScript + Tailwind CSS + React Router v6 + Axios.
- Set up `backend/` with Node.js 18 + Express + TypeScript + Axios + dotenv + cors + zod.
- Create `.env.example` with all variables: `BNOVO_UID`, `BNOVO_ACCOUNT_ID`, `BNOVO_API_BASE`, `PORT`, `NODE_ENV`, `FRONTEND_URL`, `VITE_API_BASE_URL`.
- Configure `tsconfig.json` for both packages (strict mode).
- Configure Vite dev proxy: `/api` → `http://localhost:3000` in `vite.config.ts`.
- Add `dev` scripts for both packages (frontend: `vite`, backend: `tsx watch src/index.ts`).
- Create minimal entry points: `frontend/src/main.tsx`, `backend/src/index.ts` (Express app listening on PORT).
- Set up `.gitignore` (node_modules, dist, .env, etc.).

## Tech Requirements

- Technologies: Node.js 18+, React 18+, TypeScript, Vite, Tailwind CSS, Express.
- Libraries (frontend): `react`, `react-dom`, `react-router-dom`, `axios`, `tailwindcss`, `postcss`, `autoprefixer`.
- Libraries (backend): `express`, `axios`, `dotenv`, `cors`, `zod`, `tsx` (dev).
- Dependencies: after stt-000 (fixtures and .env exist).
- Constraints: minimal boilerplate, no extra features.

## Acceptance Criteria

- `npm run dev` (or equivalent) starts both frontend and backend dev servers.
- Frontend accessible at `http://localhost:5173`, backend at `http://localhost:3000`.
- `GET http://localhost:5173/api/account` proxies to backend (can return error, but proxy works).
- TypeScript compiles without errors in both packages.
- Tailwind CSS is configured and utility classes work.
- `.env.example` contains all required variables with example values.
