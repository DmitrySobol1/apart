## Goal

Apply consistent visual styling across all pages, create reusable UI utility components, implement iframe auto-height via postMessage, and verify the widget works correctly inside an iframe.

## Details

- **Tailwind theme refinement**:
  - Clean, modern design: white background, blue accent color (similar to Bnovo widget).
  - System font stack (no external fonts).
  - Consistent spacing, typography, and color usage across all pages.
  - Ensure `tailwind.config.ts` has proper content paths.
- **Reusable components**:
  - `LoadingSpinner` (`components/LoadingSpinner.tsx`): centered spinner, configurable size.
  - `ErrorMessage` (`components/ErrorMessage.tsx`): inline error with optional "Retry" button callback.
- **iframe auto-height** via `postMessage`:
  - Widget sends `window.parent.postMessage({ type: 'resize', height: document.body.scrollHeight }, '*')` on:
    - Route changes (React Router navigation).
    - Content updates (after data loads, form validation changes, etc.).
  - Use `ResizeObserver` on `document.body` for automatic height tracking.
- **Host page example**:
  - Create a test HTML file (e.g., `test-iframe.html`) with:
    - An `<iframe>` embedding the widget.
    - Inline `<script>` listening for `message` events and adjusting iframe height.
- **Page transitions**: basic CSS transitions between pages (fade or slide).
- **Polish**: ensure no double scrollbars in iframe, proper overflow handling, consistent loading/error states.

## Tech Requirements

- Technologies: React, TypeScript, Tailwind CSS.
- Files: `frontend/src/components/LoadingSpinner.tsx`, `frontend/src/components/ErrorMessage.tsx`, `frontend/src/styles/globals.css`, `frontend/tailwind.config.ts`, `test-iframe.html`.
- Dependencies: after stt-005, stt-006, stt-007 (all pages built).
- Constraints: ≤400 LOC total across new/modified files.

## Acceptance Criteria

- All pages have consistent, polished Tailwind styling (blue accents, white background).
- LoadingSpinner and ErrorMessage components are reusable and used across pages.
- Widget loads correctly inside `<iframe>` with auto-adjusting height.
- No double scrollbars when embedded.
- `test-iframe.html` demonstrates working iframe embedding with auto-height script.
- Page transitions are smooth.
- Body overflow is handled properly for iframe context (`overflow: hidden` on body).
