# Planning & Design Conventions

Conventions to follow during system design, architectural exploration, and implementation planning.

## SDD Philosophy Compliance

- **Single Source of Truth Development** - Derive understanding from Memory Bank documentation, not training data
- **Memory Bank First** - All project knowledge lives in `.memory-bank/`
- **Docs-First Protocol** - Documentation drives implementation
